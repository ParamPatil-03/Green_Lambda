Step 1 — AWS Connection

Connect Your AWS Account

Provide read-only credentials once. Green Lambda will auto-discover Lambda functions and historical CloudWatch metrics.

🔑 How to generate your AWS Access Keys

↗ Go to AWS Console
1
Open the AWS IAM Console

Log into your AWS account and navigate to the IAM (Identity and Access Management) service. This is where all user credentials are managed.

↗ Open AWS IAM Users Console
2
Create a New IAM User

Click Create user, give it any name (e.g. greenlambda-reader), and under Permissions choose Attach policies directly. Search and attach both:
AWSLambda_ReadOnlyAccess  +  CloudWatchReadOnlyAccess

3
Generate the Access Key

Click on your newly created user → go to the Security credentials tab → scroll down and click Create access key. When prompted for use case, select "Application running outside AWS".

↗ Open Security Credentials Page
4
Copy & Paste the Keys below

AWS will show you the Access Key ID (starts with AKIA…) and the Secret Access Key. Copy both and paste them into the fields below. ⚠ The Secret Key is only shown once — save it immediately!
